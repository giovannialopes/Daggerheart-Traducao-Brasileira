# Daggerheart - Tradução Brasileira
Esse módulo aplica a tradução para o sistema Daggerheart.

# Instruções
1. Ative o módulo.
2. Selecione português do Brasil em configurações.

# Instalação

## Instalação Automática
Pesquise **Daggerheart - Tradução** em módulos. Depois instale.

## Instalação Manual
Vá em **módulos** e coloque o link abaixo. 
Link: https://raw.githubusercontent.com/giovannialopes/Daggerheart-Traducao-Brasileira/main/module.json

# Problemas na Tradução
Se encontrar problemas na tradução informe qual é o problema nesse [link](https://github.com/giovannialopes/Daggerheart-Traducao-Brasileira/issues).
